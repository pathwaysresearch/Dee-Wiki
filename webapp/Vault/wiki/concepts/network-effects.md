# Network Effects

A phenomenon where a platform becomes more valuable as more users join, which in turn increases consumers' switching costs. A phenomenon where a platform's value increases as more users join, creating a virtuous circle where more sellers attract more buyers and vice versa. The phenomenon where new users of a service bring in additional users, creating significant value in customer acquisition. A phenomenon where a platform's value increases with more users, observed in companies like Amazon and Airbnb.
